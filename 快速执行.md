# 快速执行 - 彻底删除并重建

## 在 Git Bash 中执行

```bash
# 1. 给脚本添加执行权限
chmod +x 彻底删除并重建.sh

# 2. 运行脚本
./彻底删除并重建.sh
```

## 或者手动执行（如果脚本不工作）

```bash
# 1. 完全删除 .git
rm -rf .git

# 2. 重新初始化
git init

# 3. 手动添加文件（不要用 git add .）
git add frontend/src/
git add frontend/package*.json
git add frontend/tsconfig*.json
git add frontend/vite.config.ts
git add frontend/index.html
git add frontend/eslint.config.js
git add frontend/README.md

git add backend/core/
git add backend/config/
git add backend/manage.py
git add backend/requirements.txt

git add .gitignore
git add *.md *.sh *.bat *.ps1

# 4. 检查是否有大文件被添加
git ls-files --cached | grep -E "db.sqlite3|echodraw-master|GenWanMinTW|\.ttf|\.jpg|\.png|\.webp"

# 如果有输出，手动移除：
# git rm --cached <文件名>

# 5. 提交
git commit -m "Initial commit (clean)"

# 6. 检查大小
du -sh .git/objects
```

## 找出到底是什么文件这么大

如果想先看看是什么文件占用了空间：

```bash
chmod +x 找出大文件.sh
./找出大文件.sh
```

或者直接运行：

```bash
git rev-list --objects --all | \
  git cat-file --batch-check='%(objecttype) %(objectname) %(objectsize) %(rest)' | \
  awk '/^blob/ {print substr($0,6)}' | \
  sort -k2 -n -r | \
  head -20
```
