# PWA 图标文件说明

为了完整支持 PWA 功能，您需要在 `frontend/public/` 目录下准备以下图标文件：

## 必需的图标文件

1. **icon-192.png** - 192x192 像素
   - 用于主屏幕图标和 Android 设备
   - 建议使用 PNG 格式，支持透明背景

2. **icon-512.png** - 512x512 像素
   - 用于高分辨率显示和启动画面
   - 建议使用 PNG 格式，支持透明背景

## 图标设计建议

- 使用简洁、易识别的设计
- 确保图标在小尺寸下仍然清晰可辨
- 建议使用圆角设计（系统会自动应用遮罩）
- 背景色建议使用主题色或透明

## 生成图标的方法

1. **在线工具**：
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator
   - https://favicon.io/

2. **使用设计软件**：
   - 创建 512x512 的设计
   - 导出为 PNG 格式
   - 使用图像编辑软件调整到 192x192

3. **从现有 logo 转换**：
   - 如果您已有 logo，可以使用图像编辑软件调整尺寸

## 临时解决方案

如果暂时没有图标文件，可以：
1. 使用现有的 vite.svg 转换为 PNG
2. 创建一个简单的占位图标
3. 系统会使用默认图标（可能显示为空白或浏览器默认图标）

## 注意事项

- 图标文件必须放在 `frontend/public/` 目录下
- 文件名必须与 `manifest.json` 中配置的路径完全一致
- 部署到生产环境前，请确保图标文件已正确上传
